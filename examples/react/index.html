Usage

Install the adapter and use the component anywhere in your React app.

Install

npm install @danilo.riedel/img-fit-react

Component

import { ImgFit } from '@danilo.riedel/img-fit-react';

function App() {
  return (
    <ImgFit
      src="https://cdn.example.com/photo.jpg"
      alt="Photo"
      params="f=webp&q=80"
    />
  );
}

Configuration

Pass an options prop to control DPR, debounce, and more.

Disable retina

<ImgFit
  src="https://cdn.example.com/photo.jpg"
  alt="Photo"
  options={{ dpr: false }}
/>

Request CSS pixel widths only — useful for thumbnails or low-bandwidth contexts.

Extra CDN parameters

<ImgFit
  src="https://cdn.example.com/photo.jpg"
  alt="Photo"
  params="f=webp&q=80"
/>
{/* Result: ?f=webp&q=80&rs=1250 */}

The params string is appended to the URL before ?rs=.

Resize debounce

<ImgFit
  src="https://cdn.example.com/photo.jpg"
  alt="Photo"
  options={{ resizeDebounceMs: 200 }}
/>

Default: 100 ms. Increase to reduce URL updates during rapid window resizing.

How it works

  • The component renders a regular <img> element.
  • On mount it measures the rendered container and computes the optimal width.
  • Width is multiplied by devicePixelRatio (DPR) before snapping to cache-friendly breakpoints.
  • It appends ?rs=<width> to the source URL automatically.
  • A ResizeObserver keeps the source in sync when the container resizes.

Part of the img-fwd family.
