How Sizing Works

img-fit measures the rendered container width and snaps it up to the next cache-friendly breakpoint. Exact widths would fragment caches, so every value is rounded using the following grid:

  • < 300 px → rounded to the next 20 px
  • 300 px – 1000 px → rounded to the next 50 px
  • > 1000 px → rounded to the next 250 px

On retina screens the measured width is multiplied by window.devicePixelRatio and then snapped again, so images stay sharp without requesting unnecessary pixels.

Usage

Add a data attribute to your HTML and initialize img-fit in JavaScript.

HTML

<img
  data-img-fit="https://cdn.example.com/photo.jpg"
  alt="Photo"
/>

JavaScript

import ImgFit from 'img-fit';

ImgFit.init();

// Optional: customize options
ImgFit.init({
  dpr: true,
  observeResize: true
});
