Usage

Add a data attribute to your HTML and initialize img-fit once in JavaScript.

HTML

<img
  data-img-fit="https://cdn.example.com/photo.jpg"
  alt="Photo"
/>

JavaScript

import ImgFit from '@danilo.riedel/img-fit';

// Init with defaults — DPR on, resize observed
ImgFit.init();

// Or pass options
ImgFit.init({
  dpr: true,
  observeResize: true,
  resizeDebounceMs: 100
});

Configuration

All options can be passed to ImgFit.init() globally or per element via ImgFit.watch().

Disable retina

ImgFit.init({ dpr: false });

Requests CSS pixel widths only — useful when the CDN should not serve retina-quality images (e.g. thumbnails where the file size saving is not worth it).

Extra CDN parameters

<img
  data-img-fit="https://cdn.example.com/photo.jpg"
  data-img-fit-params="f=webp&q=80"
  alt="Photo"
/>
<!-- Result: ?f=webp&q=80&rs=1250 -->

Appended to the URL before ?rs= is added.

Resize debounce

ImgFit.init({
  resizeDebounceMs: 200
});

Default: 100 ms. Increase to reduce URL updates during rapid window resizing.

Disable lazy recompute

ImgFit.init({ lazy: false });

Skips the IntersectionObserver pass for elements with loading="lazy". Use when you know all images are in the viewport on load.

Custom measurement base

<div class="card">
  <img
    data-img-fit="https://cdn.example.com/photo.jpg"
    data-img-fit-base=".card"
    alt="Photo"
  />
</div>

Use data-img-fit-base with a CSS selector to pick which ancestor to measure.

Programmatic watch

import ImgFit from '@danilo.riedel/img-fit';

const el = document.querySelector('#hero-img');
ImgFit.watch(el, {
  url: 'https://cdn.example.com/photo.jpg',
  dpr: false
});

Watch a single element with a URL supplied in JavaScript rather than via a data attribute.
