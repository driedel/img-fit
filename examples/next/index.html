Next.js adapter for img-fwd

Next.js next/image
Vanilla Example All Examples

Usage

Install the adapter and wrap your images with the img-fit component.

Install

npm install @img-fit/next

Component

import { ImgFitImage } from '@img-fit/next';

export default function Page() {
  return (
    <div style={{ position: 'relative', aspectRatio: '16 / 9' }}>
      <ImgFitImage
        src="https://cdn.example.com/photo.jpg"
        alt="Photo"
        params="f=webp&q=80"
        fill
      />
    </div>
  );
}

How it works

  • The component wraps next/image using the fill prop.
  • On the client it measures the parent container and builds the optimal URL.
  • It appends ?rs=<width> and passes the result to next/image.
  • SSR stays stable — the URL is updated after hydration.

Part of the img-fwd family.
