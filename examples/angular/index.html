Usage

Install the adapter and apply the directive to any image element.

Install

npm install @danilo.riedel/img-fit-angular

Component

import { ImgFitDirective } from '@danilo.riedel/img-fit-angular';

@Component({
  standalone: true,
  imports: [ImgFitDirective],
  template: `
    <img
      [imgFit]="'https://cdn.example.com/photo.jpg'"
      imgFitParams="f=webp&q=80"
      alt="Photo"
    />
  `
})
export class MyComponent {}

Configuration

Use [imgFitOptions] to control DPR, debounce, and more.

Disable retina

<img
  [imgFit]="url"
  [imgFitOptions]="{ dpr: false }"
  alt="Photo"
/>

Request CSS pixel widths only — useful for thumbnails or low-bandwidth contexts.

Extra CDN parameters

<img
  [imgFit]="url"
  imgFitParams="f=webp&q=80"
  alt="Photo"
/>
<!-- Result: ?f=webp&q=80&rs=1250 -->

The imgFitParams string is appended to the URL before ?rs=.

Resize debounce

<img
  [imgFit]="url"
  [imgFitOptions]="{ resizeDebounceMs: 200 }"
  alt="Photo"
/>

Default: 100 ms. Increase to reduce URL updates during rapid window resizing.

How it works

  • The imgFit directive attaches to a native <img>.
  • It measures the rendered container after the view initializes.
  • Width is multiplied by devicePixelRatio (DPR) before snapping to cache-friendly breakpoints.
  • It appends ?rs=<width> to the source URL automatically.
  • A ResizeObserver keeps the source in sync when the container resizes.

Part of the img-fwd family.
